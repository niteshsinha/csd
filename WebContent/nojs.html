No java script No Fun

Hi

If you are reading this then it means that JavaScript is not enabled on your browser. Our site has a lot of featurs that require javascript in order to function properly. So we requist you to please enable it in order to enjoy all features of our site.

If you are not sure how to enable Java Script here is a small How To for a few popular browsers:

Firefox 2.x.x and 3.x.x
  • Open Firefox.
  • On the Tools menu, click Options.
  • Click on the Content icon.
  • Check the box next to Enable JavaScript.
  • Click Ok
Firefox 1.x.x
  • Open Firefox.
  • On the Tools menu, click Options.
  • Click Web Features in the Options list.
  • Under the Web Features section, check the box next to Enable JavaScript.
  • Click Ok
Microsoft Internet Explorer 5 or Later
  • Open Internet Explorer.
  • On the Tools menu, click Internet Options.
  • On the Security tab, click Internet.
  • Click Custom Level.
  • Scroll down to Active scripting.
  • Click Enable (or Prompt).
  • Click OK.
  • Click Yes.
  • Click OK.
Opera 8.x.x and 9.x.x
  • Open Opera.
  • On the Tools menu, click Preferences.
  • Click Content in the Preferences list.
  • Check the box next to Enable JavaScript.
  • Click the JavaScript Options button to open the JavaScript Options box.
  • Check the boxes that you want to allow.
  • Click OK.
Netscape 7
  • Open Netscape.
  • On the Edit menu, click Preferences.
  • Double click Advanced in the Category list.
  • Select Scripts & Plugins from the menu.
  • Under the Enable JavaScript for section, check the box next to Navigator.
  • Check the boxes under Allow scripts to section that you want to allow.
  • Click OK.
Safari
  • Open Safari
  • On the Safari menu, click on Preferences.
  • Click the Security icon.
  • Click on Enable JavaScript next to the Web Content section if it is not checked.
  • Close the Preferences window.
  • Close and restart Safari.
